# Talend Studio

Talend Studio is the primary development environment for building data processes.

## Key Capabilities
- Visual drag-and-drop job design
- Built-in debugging and validation tools
- Local and remote execution modes
- Rich component library
- Repository-based metadata management

## Repository Structure
- Metadata – connections, schemas, APIs
- Jobs – ETL and integration processes
- Routines – reusable helper functions
- Contexts – environment variables (DEV / TEST / PROD)

Talend Studio integrates directly with Git for version control
and with TAC for deployment and execution.
