# Talend Administration Center (TAC)

TAC is the central management and orchestration layer of Talend.

## Core Responsibilities
- Scheduling and execution of Jobs
- User and permission management
- Environment and context selection
- Execution monitoring and logging
- Alerting and failure tracking

## Monitoring Capabilities
- Real-time execution status
- Historical run analysis
- Log inspection
- Failure diagnostics and alerts

TAC interacts with Nexus for artifact retrieval
and with Job Servers for runtime execution.
